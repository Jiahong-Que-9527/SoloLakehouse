# SoloLakehouse 面试级教程总入口

本教程面向“平台工程师高强度面试”准备，目标是让你可以在不同深度下稳定讲清：

- 项目是什么，解决什么问题
- 为什么这样设计，而不是别的方案
- 关键实现如何落地（代码路径、运行路径、故障路径）
- 如果升级到 v2/v3，你会如何做架构演进

## 建议学习顺序

1. `00-总览与学习路线.md`
2. `01-项目定位与面试叙事.md`
3. `02-整体架构与八层目标.md`
4. `03-环境与部署全流程.md`
5. `04-数据流与Medallion详解.md`
6. `05` 到 `12`（代码与工程体系）
7. `13` 到 `18`（决策、演进、面试话术与冲刺）
