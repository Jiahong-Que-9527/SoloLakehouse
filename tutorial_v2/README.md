# SoloLakehouse v2 面试级教程（平台工程师 100k+）

这是一套专门面向 **v2（Dagster Orchestrated Platform）** 的中文深度教程，目标不是“会用”，而是“能讲透、能追问、能落地”。

你可以把它理解为：

- 技术事实层：代码到底做了什么（源码映射）
- 架构表达层：为什么这么做（设计取舍）
- 面试输出层：如何讲清楚（30秒/2分钟/5分钟模板）
- 运营排障层：出问题怎么查（实战手册）

## 学习产出目标（验收标准）

完成后你应能稳定做到：

- 10分钟讲清楚“从 v1 到 v2 的演进价值”
- 20分钟源码级解释 Dagster 资产/调度/传感器/检查
- 30分钟应对平台工程师深挖（可靠性、可运维性、发布治理）
- 面试官追问“为什么不是 Airflow/Prefect”时给出有边界的决策说明
- 面试官追问“生产化下一步”时给出 v3/v4 的可执行路线

---

## 使用方式

- 每篇都按统一结构写：**结论先行 → 代码事实 → 设计取舍 → 面试表达**。
- 推荐每篇至少做一次“口述复盘”：
  - 第一轮：看文档讲
  - 第二轮：不看文档讲
  - 第三轮：针对追问补短板
